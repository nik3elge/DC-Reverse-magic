Минималистичный веб-инструмент для автоматической генерации «обратных заклинаний» в стиле Затанны Затары из вселенной DC Comics. Проект служит удобной утилитой для перевода и локализации комиксов DC, помогая быстро адаптировать заклинания без необходимости переворачивать слова вручную.

## ✨ Особенности

*   **Умный реверс:** Переворачивает только буквы в словах, сохраняя исходный порядок самих слов.
*   **Сохранение регистра:** Заглавные буквы остаются на своих визуальных позициях (например, `Магия` -> `Яигам`).
*   **Игнорирование пунктуации:** Запятые, точки, дефисы и пробелы остаются на своих законных местах.
*   **Копирование в один клик:** Готовое заклинание копируется в буфер обмена простым нажатием на результат.
